# 태영의 인더스트리얼 디자인 마스터클래스 🎨

**안티그래피(Antigravity)를 활용하여 구축된 개인 웹사이트 및 포트폴리오 프로젝트입니다.**

이 프로젝트는 인더스트리얼 디자인(산업 디자인)의 핵심 철학과 실무 노하우를 공유하고, 개인의 디자인 포트폴리오를 매력적으로 선보이기 위해 기획되었습니다. 모던하고 감각적인 UI/UX를 통해 방문자에게 직관적이고 아름다운 웹 경험을 제공합니다.

## ✨ 프로젝트 개요

- **주제**: 인더스트리얼 디자인 포트폴리오 및 마스터클래스 (온라인 강의/아티클)
- **목적**: 
  - 개인 작업물 및 디자인 프로젝트 전시
  - 디자인 프로세스와 철학에 대한 인사이트 공유
  - 예비 디자이너들을 위한 마스터클래스 형식의 교육 콘텐츠 제공
- **주요 기술**: HTML, CSS, JavaScript, Antigravity (AI Coding Assistant)

## 🎯 핵심 기능 및 구성

1. **포트폴리오 쇼케이스 (Portfolio)**
   - 3D 모델링, 제품 디자인, 렌더링 등 다양한 인더스트리얼 디자인 작업물을 고해상도 이미지와 함께 전시합니다.
   - 각 프로젝트의 기획 의도, 문제 해결 과정, 최종 결과물을 스토리텔링 형식으로 풀어냅니다.

2. **마스터클래스 (Masterclass)**
   - 아이디에이션부터 최종 프로토타이핑까지의 디자인 프로세스를 단계별로 설명합니다.
   - 제품 디자인 실무에서 활용되는 팁과 툴(Rhino, Fusion 360, KeyShot 등) 활용 노하우를 제공합니다.

3. **디자인 철학 & 소개 (About)**
   - 디자이너 '태영'의 디자인 철학과 비전, 그리고 그간의 발자취를 소개합니다.

## 🛠️ 개발 주안점

- **심미성 (Aesthetics)**: 인더스트리얼 디자인의 특성에 걸맞게 미니멀하면서도 세련된 타이포그래피와 레이아웃을 활용하며, 다크 모드 등 프리미엄한 감성을 전달합니다.
- **동적 상호작용 (Interactive Design)**: 마이크로 인터랙션과 부드러운 트랜지션 애니메이션을 통해 생동감 있는 사용자 경험을 구현합니다.
- **반응형 웹 (Responsive Web)**: 데스크탑부터 모바일 기기까지 다양한 해상도에 완벽하게 대응하는 레이아웃을 제공합니다.
